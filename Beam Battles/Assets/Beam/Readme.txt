Thank you for downloading the Beam Interactive SDK. The SDK includes:
* Examples - Example scenes and instructions that demonstrate the functionality in this SDK.
* Prefabs - The BeamManager prefab which you can use to talk to the Beam Service
* Source - The DLLs, helper scripts and editor extensions for the SDK

There are instructions included in each of the example scenes.

You can learn more about the Beam SDK at: https://dev.beam.pro/

For bugs or other issues, you can find support at the official GIT repo: https://github.com/WatchBeam/interactive-unity-plugin/issues

For feedback, you can visit our feedback site: https://feedback.beam.pro/forums/382521-lazers-and-shiny-things
